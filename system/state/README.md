# Captured Machine State

Files in this directory are generated by `./setup.sh capture`.

- `apt-manual.txt` records the current `apt-mark showmanual` output for reference
- `flatpak-apps.txt` records current user Flatpak app IDs and can be reused during restore
- `system-info.txt` records the capture timestamp and host OS details

Review the generated files before committing them.
